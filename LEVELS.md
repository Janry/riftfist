# Рівні — дизайн-документ (жива чернетка)

Тут описуємо **потенційні рівні** гри. Рівні прив'язані до серій Rick and Morty
(починаємо з 1 сезону). Це чернетка для ітерацій — додаємо/міняємо вільно.

> **Арт:** процедурний фон у прототипі — лише ЗАГЛУШКА під механіку. Фінальні
> задники = реальні мальовані ассети (підключаються шаром із паралаксом, як спрайти героя).
>
> **Механіка рівня (готова):** intro → хвиля → «вперед ~пів екрана» → лок арени →
> наступна хвиля → … → міні-бос → бос → зачищено. Кожен рівень = набір арен.

**Шаблон полів:** Серія · Тематика · Атмосфера · Локації/арени · Хто поганий (моби) ·
Хто хороший · Хто допомагає · Хто заважає · Головний бос · Ідеї механік/хазардів · Арт-напрям.

Статуси: 🟡 концепт · 🟠 в роботі · 🟢 готово

## Рівень 1 — «За насінням» 🟡
**Серія:** S1E1 «Pilot» (Вимір 35-C, Мега-насіння).
**Холодний відкривач:** гра починається в **розбитій тарілці Ріка, зібраній із мотлоху й завеленій бляшанками** з-під пива (за пілотом: п'яний Рік, «нейтринна бомба», нічне небо). Це туторіальна арена.

| Аспект | Деталь |
|---|---|
| Тематика | Старт у **розбитій літаючій тарілці** (мотлох + гора бляшанок/пляшок) → портал → чужий вимір по Мега-насіння |
| Атмосфера | П'яний-геній хаос на старті → дикий чужий ландшафт (Мега-дерева), тривога + абсурд |
| Локації/арени | (1) **кабіна розбитої тарілки** серед бляшанок — туторіал руху/удару; (2) портал-перехід; (3) чужий вимір — поля Мега-дерев; (4) інтервимірна «митниця» |
| Хто поганий (моби) | Дрібні інопланетні стражі-комахи; митні охоронці-бюрократи |
| Хто хороший | Рік, Морті |
| Хто допомагає | Рік (портал-ган, ґаджети, підбір зброї); Морті носить насіння |
| Хто заважає | Незграбність Морті; митниця (перевірки); **бляшанки/пляшки** як перешкоди й снаряди на підлозі |
| **Головний бос** | **Велетенський інопланетний страж Мега-дерев** — те саме «нечітке створіння», що напало в пілоті |
| Ідеї механік/хазардів | Слизькі калюжі бухла (ковзання) у тарілці; копаєш/жбурляєш бляшанки; «контрабанда»: нести насіння = тимчасово без зброї; митні турнікети-ворота |
| Арт-напрям | Брудний sci-fi гараж: іржава тарілка з мотлоху, гора зелених/срібних бляшанок, тріснуте скло кабіни, зелене сяйво портал-гана, нічне небо → психоделічні фіолетово-зелені поля чужого виміру |

**Арт-референс (за пілотом, для задника арени 1):**
- Кабіна саморобної тарілки: панель приладів зі сміття/скотчу, тріснутий ковпак-купол, миготливі лампочки.
- Підлога завалена **бляшанками й пляшками** (купи, що перекочуються), калюжі.
- Знадвору крізь купол — нічне небо/вулиця передмістя; поряд — зелений овал **порталу**.
- Палітра: брудно-зелене/сіро-сталеве залізо, бурштинові лампочки, отруйно-зелений неон порталу.

## Рівень 5 — «Meeseeks and Destroy» 🟡
**Серія:** S1E5 «Meeseeks and Destroy» (пригода Морті: бобове стебло, велетні, суд + B-сюжет Місіксів).
**Фішка рівня:** майданчик для **нестандартних швидких сегментів** (підйом/спуск у дусі Battletoads), а не лише бійка на аренах.

| Аспект | Деталь |
|---|---|
| Тематика | Морті веде «казкову» пригоду (пародія на «Джек і бобове стебло»): підйом до світу велетнів, місто/бар, убивство велетня → суд → втеча. Паралельно вдома — коробка **Містера Місікса**, що множиться й шаленіє |
| Атмосфера | Казка-перевертень: світла фентезі-зав'язка → жорстка реальність і абсурд; «існування — біль» |
| Локації/арени | (1) **ПІДЙОМ по бобовому стеблу** [нестандартний сегмент]; (2) місто/бар велетнів — бійка; (3) гігантський **суд** — сет-піс/бос; (4) **СПУСК-втеча вниз** [нестандартний сегмент] |
| Хто поганий (моби) | Велетні-городяни, бар-вишибали, судові пристави-велетні; (B) маніакальні Місікси, що множаться |
| Хто хороший | Морті (лідер пригоди), Рік, Саммер |
| Хто допомагає | Рік (ґаджети/портал); «нормальний» Місікс на одне просте завдання (бафф/підказка) |
| Хто заважає | Рій Місіксів (B-сюжет, тиск-таймер); судова бюрократія велетнів; хазарди на стеблі/спуску |
| **Головний бос** | **Велетень** (чемпіон/король у суді). **Міні-бос:** **Містер Місікс** — лідер ошалілого рою |
| Ідеї механік/хазардів | **Підйом:** вертикальний скрол-клайм, хапаєш ліани/виступи, збиваєш ворогів-верхолазів, ухиляєшся від падаючого каміння. **Спуск:** швидкий сегмент падіння/слайду — ухиляння на швидкості (Battletoads-tower). **Рій Місіксів:** не зачистив швидко → множаться (тиск на темп); «existence is pain» |
| Арт-напрям | Соковите фентезі-небо й стебло у хмарах → масивне кам'яне місто/суд велетнів (усе завелике, гравець малий); сині Місікси-спалахи |

## Рівень 6 — «Raising Gazorpazorp» 🟡
**Серія:** S1E7 «Raising Gazorpazorp» (планета Ґазорпазорп: дикі самці vs хай-тех самиці; Морті-молодший).
**Фішка рівня:** два контрастні світи + **companion-«виховання»** як механіка.

| Аспект | Деталь |
|---|---|
| Тематика | Морті стає батьком напів-ґазорпіанця (**Морті-молодший**); Рік і Саммер летять на **Ґазорпазорп** — войовничі самці на пустці проти гіпер-розвиненої матріархальної цивілізації самиць; сатира на гендер |
| Атмосфера | Mad-Max руда пустка (самці) ↔ стерильна біло-рожева хай-тех утопія (самиці); абсурд + сатира |
| Локації/арени | (1) інтервимірний ломбард/мол (купівля «Ґвендолін») — короткий вступ; (2) **пустка Ґазорпазорпа** — орди самців, бійки; (3) аварія/полон; (4) **хай-тех місто самиць**; паралельно Земля — Морті-молодший росте/бунтує |
| Хто поганий (моби) | Дикі самці-ґазорпіанці (дубини/списи, верхи на тварюках); вартові-самці |
| Хто хороший | Рік, Саммер, Морті |
| Хто допомагає | Цивілізація **самиць** (хай-тех, апгрейд/вихід); Морті-молодший — коли подорослішає, стає союзником |
| Хто заважає | Орди самців; полон/протокол міста самиць; **буйний Морті-молодший** (поки малий — трощить усе підряд) |
| **Головний бос** | **Вожак-чемпіон самців Ґазорпазорпа** (велетенський войовник на пустці). *Альт-фінал:* «Верховна Лідерка» самиць як перемовний бос (не бійка) |
| Ідеї механік/хазардів | **«Виховання» (companion):** Морті-молодший росте по ходу рівня — спершу заважає (б'є все підряд), потім еволюціонує в помічника-ескорт; **їзда на тварюці** — короткий vehicle-сегмент (як speeder у Battletoads); контраст-арена в місті самиць (стелс/без бійки); тотеми-ідоли як руйнівні хазарди |
| Арт-напрям | Руда пустка з кістками й примітивними тотемами (самці) ↔ гладкий біло-рожевий хай-тек (самиці); Морті-молодший — зелено-шкірястий напівінопланетянин, що росте по сценах |

## Рівень 8 — «Mortynight Run» 🟡
**Серія:** S2E2 «Mortynight Run» (рятуємо газову істоту, втеча від Федерації, аркада «Roy»). *Старт Сезону 2.*
**Фішка рівня:** ескорт-«втеча» + нестандартний арт-сегмент «Roy: A Life Well Lived»; моральний твіст-фінал.

| Аспект | Деталь |
|---|---|
| Тематика | Морті рятує телепатичну газову істоту («**Fart**») від найманця → втеча через ворожі зони, щоб повернути її в її вимір; фінал — важкий моральний вибір |
| Атмосфера | Неонова космо-погоня + аркадний абсурд → гірко-меланхолійний фінал («Goodbye Moonmen») |
| Локації/арени | (1) угода Ріка з найманцем + звільнення Fart; (2) **втеча-ескорт** під обстрілом Федерації; (3) інопланетна **аркада** — схованка, VR-сегмент **«Roy»** + погоня між автоматами; (4) фінал-портал + **твіст-вибір**; (бонус) Jerryboree — комічна рамка |
