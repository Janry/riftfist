# Рівні — дизайн-документ (жива чернетка)

Тут описуємо **потенційні рівні** гри. Рівні прив'язані до серій Rick and Morty
(починаємо з 1 сезону). Це чернетка для ітерацій — додаємо/міняємо вільно.

> **Арт:** процедурний фон у прототипі — лише ЗАГЛУШКА під механіку. Фінальні
> задники = реальні мальовані ассети (підключаються шаром із паралаксом, як спрайти героя).
>
> **Механіка рівня (готова):** intro → хвиля → «вперед ~пів екрана» → лок арени →
> наступна хвиля → … → міні-бос → бос → зачищено. Кожен рівень = набір арен.

**Шаблон полів:** Серія · Тематика · Атмосфера · Локації/арени · Хто поганий (моби) ·
Хто хороший · Хто допомагає · Хто заважає · Головний бос · Ідеї механік/хазардів · Арт-напрям.

Статуси: 🟡 концепт · 🟠 в роботі · 🟢 готово

## Рівень 1 — «За насінням» 🟡
**Серія:** S1E1 «Pilot» (Вимір 35-C, Мега-насіння).

| Аспект | Деталь |
|---|---|
| Тематика | Старт: інтер'єр **літаючої тарілки** Ріка, завалений банками й пляшками з-під бухла; портал → чужий вимір по Мега-насіння |
| Атмосфера | П'яний-геній хаос на старті → дикий чужий ландшафт (Мега-дерева), тривога + абсурд |
| Локації/арени | (1) тарілка/гараж — туторіал руху й удару; (2) портал-перехід; (3) чужий вимір — поля Мега-дерев; (4) інтервимірна «митниця» |
| Хто поганий (моби) | Дрібні інопланетні стражі-комахи; митні охоронці-бюрократи |
| Хто хороший | Рік, Морті |
| Хто допомагає | Рік (портал-ган, ґаджети, підбір зброї); Морті носить насіння |
| Хто заважає | Незграбність Морті; митниця (перевірки); самі банки/пляшки як перешкоди на підлозі |
| **Головний бос** | **Велетенський інопланетний страж Мега-дерев** — те саме «нечітке створіння», що напало в пілоті |
| Ідеї механік/хазардів | Слизькі калюжі бухла (ковзання) у тарілці; «контрабанда»: нести насіння = тимчасово без зброї; митні турнікети-ворота |
| Арт-напрям | Брудний sci-fi гараж (зелені портали, неон-залізо) → психоделічні фіолетово-зелені поля чужого виміру |

## Рівень 2 — «Сон Голденфолда» 🟡
**Серія:** S1E2 «Lawnmower Dog» (інцепція у сни вчителя математики).

| Аспект | Деталь |
|---|---|
| Тематика | Залазимо у **сон містера Голденфолда**, щоб він поставив Морті залік; шари снів, кошмарна логіка |
| Атмосфера | Сюрреал, мінлива геометрія сну, шкільна тривога як хорор-комедія |
| Локації/арени | (1) клас-кошмар Голденфолда; (2) «сексі-сон» — абсурдні захисники підсвідомості; (3) глибший шар — світ **Лякливого Террі** |
| Хто поганий (моби) | Конструкти сну, кошмарні «учні», захисники підсвідомості Голденфолда |
| Хто хороший | Рік, Морті, Саммер |
| Хто допомагає | **Лякливий Террі** — після «ночівлі» в його сні стає союзником; Рік (вихід зі сну) |
| Хто заважає | Нелогіка сну (зміна правил/гравітації); сам Голденфолд із пістолетом уві сні |
| **Головний бос** | **Лякливий Террі** (Scary Terry) — «you can run but you can't hide!»; після перемоги/дружби — перехід |
| Ідеї механік/хазардів | Шари-сни (зміна локації «провалом» углиб); зони, де правила інверсні; погоня-сегмент від Террі |
| Арт-напрям | Неон-кошмар класу, що плавиться; червоно-чорний світ Террі (оммаж Фредді Крюгеру) |

## Рівень 3 — «Анатомічний парк» 🟡
**Серія:** S1E3 «Anatomy Park» (парк розваг усередині тіла бездомного Рубена).

| Аспект | Деталь |
|---|---|
| Тематика | Зменшені, всередині **людського тіла**; парк атракціонів по органах; пародія на «Парк Юрського періоду», але з хворобами |
| Атмосфера | Біо-хорор, гидко-смішно, занедбаний парк, що руйнується разом із вмираючим тілом |
| Локації/арени | (1) легені/бронхи; (2) кишківник; (3) серце/судини; (4) печінка — лігво боса |
| Хто поганий (моби) | Хвороби-монстри: Бубонна чума, Туберкульоз, Гонорея, E. coli; Гепатит C (суперник) |
| Хто хороший | Морті (зменшений), Доктор Ксенон Блум, Енні, Пончо, Роджер |
| Хто допомагає | **Доктор Ксенон Блум** — гід/союзник, пояснює слабкі місця; Енні (бойова підтримка) |
| Хто заважає | Паніка персоналу (Пончо зраджує); тіло вмирає → середовище деградує (таймер) |
| **Головний бос** | **Гепатит А** — велетенський монстр-хвороба, кульмінаційна загроза парку |
| Ідеї механік/хазардів | Таймер «тіло вмирає» (тиск на темп); потоки крові (несе гравця); імунні клітини як нейтральні/перешкоди |
| Арт-напрям | Вологі органічні тунелі (рожево-червоне, пульсація), іржаві атракціони парку поверх плоті |

## Нотатки
- «Хто заважає» ділимо на: *середовищні хазарди* (дані/тригери — калюжі, таймери, потоки)
  та *персонажів-перешкод* (міні-боси/нейтрали).
- Один рівень може бути кількома вимірами/локаціями (арени всередині рівня).
- Імена/боси — прямі з серіалу (рішення власника: використовуємо реальний R&M).
